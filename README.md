# Crawling the Homepage of Gatech CoC Faculty
A simple crawler to crawl the home page url of Gatech CoC faculty. (https://www.cc.gatech.edu/people/faculty)

```
node version v8.12.0
npm version 6.4.1
```

### Installation

```
npm install
node app.js
```
